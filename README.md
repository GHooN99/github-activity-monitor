# GitHub Activity Monitor

## 개발 환경

이 프로젝트는 다음과 같은 도구를 사용합니다:

- TypeScript: 타입 안전한 JavaScript 개발
- tsup: TypeScript 빌드 및 번들링
- Vitest: 유닛 테스트

## 스크립트

```bash
# 의존성 설치
npm install

# 개발 모드 실행 (파일 변경 감지)
npm run dev

# 프로덕션 빌드
npm run build

# 테스트 실행
npm run test

# 테스트 감시 모드
npm run test:watch
```

Testing pre-commit hook.
